This was the first project I worked on during my internship. The website was designed to showcase a responsive and modern property listing. The UI/UX design was provided by the design team, and my responsibility was to implement it into a functional website using the Tailwind CSS framework.

Template Restaurant

This Restaurant Website was developed using Tailwind CSS to deliver a modern, responsive, and user-friendly interface. Key features include a hero section, categorized menu display (entree, dessert, cake), dynamic navigation, and an interactive slider/swiper to showcase featured dishes. The website also features an informative footer with social media links, and is fully optimized for consistency across all screen sizes (mobile, tablet, desktop).

Webtoon RealEstate

This project is a Webtoon/Comic website built with Laravel, featuring a chapter list for each comic, a comic reading page, and a user login system. It includes a backend for managing comics, chapters, and users. With authentication, users can access content based on their roles, while admins can add, edit, and delete comics or chapters through the dashboard.
